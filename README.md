This directory contains data for use with the ddq scheme. 

It includes:
- frequencies and weights for spectral integration ( https://github.com/pczarnecki/ddq-data-paper)
- gas absorption coeffients (https://github.com/ravikiran-hegde/fax) 
- ARTS XFIT, and solar source (https://github.com/atmtools/arts/) interpolated to ddq frequency grid
- MT_CKD self and foreign water vapour coeffecients (https://github.com/AER-RC/MT_CKD_H2O) interpolated to ddq frequency grid
- Longwave and Shortwave cloud optics, in the RRTMGP structure and using the RRTMGP parameterizations interpolated to the DDQ frequency grid.
